# Todo App

Create a todo app with a following functionalities
 - [ ] Show lists of todos
 - [ ] Add a todo
 - [ ] Update a todo
 - [ ] Delete a todo
 - [ ] set todo as completed
 - [ ] filter todo by status
 - [ ] clear all todos
